<b>Google Text-to-Speech (gTTS)</b> <br>
The project includes Google Colab notebook demonstrating gTTS API converting both direct string inputs and external text files into MP3 audio.

<br>
<b>Prerequisites</b> <br>
Before executing the notebook, ensure you have a text file named `sample.txt` uploaded to your working directory. The script is designed to read this specific file to generate localized speech output.

<br><br>
For comprehensive details on the library, parameter limits, and supported language codes, refer to the official documentation:
[https://gtts.readthedocs.io/en/latest](https://gtts.readthedocs.io/en/latest)
